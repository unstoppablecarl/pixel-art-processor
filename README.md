# Pixel Art Processor

Apply effects to pixel art

## Setup
##### Install
`$ pnpm install`

##### Develop
`$ pnpm dev`

##### Build
`$ pnpm build`


## Adding A New Step
Add Component to: `/src/components/Step/MyNewStep.vue`
Add to `STEP_DEFINITIONS` in `/src/steps.ts` 

## Adding A Step Data Type
Add to `STEP_DATA_TYPES` in `/src/steps.ts`
